### es6 语法

关于export的含义:
```javascript
    export function Func () { }
    import { Func } from 'func'

    export default function Func () { }
    import Func from 'func'
```

对象缩写:
```javascript
var a = 1, b = 2
{a, b} // {a: 1, b: 2}
```
